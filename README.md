# Crabba 🦀

A next-generation, decentralized social platform bridging Web2 ease-of-use with Web3/Matrix protocol power.

## Features

- **Custom VIP Invites**: Secure, invite-only access system to manage beta onboarding with usage limits and custom access codes.
- **Global Matrix Timeline**: Interact with a unified, real-time feed powered by the robust Matrix protocol.
- **SEO Optimized Profiles**: Fast, server-side rendered profiles that look great and load metadata correctly when shared on social media.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Database & Auth**: Supabase Auth/DB
- **Protocol**: Matrix Protocol
- **Styling**: Tailwind CSS, Shadcn UI

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
